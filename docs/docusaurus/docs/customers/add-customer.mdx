All Customers require a Customer Id (a.k.a Common User Id or Common Id).
This should be a unique identifier, for example their email address, user Id, hashed email etc.

To add a new Customer

<Tabs
  defaultValue="js"
  values={[
    { label: 'JavaScript', value: 'js', }
  ]
}>

<TabItem value="js">

```js
const cherry = require("cherry.ai");

cherry.customers
  .createOrUpdateCustomerAsync({
    token: "Your access token",
    customer: {
      customerId: "test-user@testing1234.org",
      name: "Jonny Goldfish",
    },
  })
  .then((u) => {
    console.log("created a Customer in Cherry");
    console.log(u);
  })
  .catch((e) => {
    console.log("An error occurred");
    console.log(e);
  });
```
