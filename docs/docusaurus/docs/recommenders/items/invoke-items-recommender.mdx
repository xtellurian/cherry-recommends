# Invoke an Item Recommender

Once a Item Recommender has been created, you can invoke it (i.e. get a recommendation) like so.

<Tabs
  defaultValue="js"
  values={[
    { label: 'JavaScript', value: 'js', }
  ]}>

<TabItem value="js">

```js
const signalbox = require("signalbox.js");

signalbox.itemsRecommenders
  .invokeItemsRecommenderAsync({
    id: "recommender_id_123", // the recommender Id
    input: {
      commonUserId: "The User Id", // the user Id to recommend for,
      arguments: {},
    },
    token: "Your access token",
  })
  .then((r) => {
    console.log("Got a recommendation");
    console.log(r);
  })
  .catch((e) => {
    console.log("Server returned an error");
    console.log(e);
  });
```

To explicitly link outcomes of recommendations to events, you can use the `correlatorId` value in the Recommendation response.
