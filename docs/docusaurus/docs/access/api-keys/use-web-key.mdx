<TabItem value="js">

To set the default API key for all requests, import cherry.ai and use the setDefaultApiKey method.

```js
const cherry = require("cherry.ai");
cherry.setDefaultApiKey("YOUR_API_KEY");
```

</TabItem>
</Tabs>

:::info Web API Key Limitations

Web API Keys can only produce events and invoke recommenders.

:::
