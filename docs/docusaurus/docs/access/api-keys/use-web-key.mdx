<TabItem value="js">

To set the default API key for all requests, import signalbox and use the setDefaultApiKey method.

This method was introduced in signalbox.js@1.5.3

```js
const signalbox = require("signalbox.js");
signalbox.setDefaultApiKey("YOUR_API_KEY");
```

</TabItem>
</Tabs>

:::info Web API Key Limitations

Web API Keys can only produce events and invoke recommenders.

:::
