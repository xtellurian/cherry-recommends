## Common Id

A Common Id is an identifer used for uniquely identifying various entities.


:::note Common Id Limitations

* A Common Id must be unique within an entity type (e.g. items, recommenders)
* A Common Id must be at least 4 characters long
* A Common Id must only contain alphanumeric characters, underscores, and hyphens

:::

The [Common User Id](./common-user-id) is a special class of Common Id.
