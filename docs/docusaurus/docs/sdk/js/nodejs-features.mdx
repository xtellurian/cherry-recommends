# Customer Features (Node JS)

[This sample (GitHub)](https://github.com/four2-ai/samples/tree/main/node-features) 
shows how to access the feature values associated with Customers


Use this sample in scenarios where you want to extend or modify existing processes using the features generated by Cherry Recommends.

Scenarios include:
* Automating email campaigns based on Health Score
* Aggregating feature data into custom reporting tools
* Extending CRM data
