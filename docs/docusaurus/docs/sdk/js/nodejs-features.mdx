# Customer Metrics (Node JS)

[This sample (GitHub)](https://github.com/four2-ai/samples/tree/main/node-features) 
shows how to access the Customer Metric values associated with Customers


Use this sample in scenarios where you want to extend or modify existing processes using the Metrics generated by Cherry Recommends.

Scenarios include:
* Automating email campaigns based on Health Score
* Aggregating metric data into custom reporting tools
* Extending CRM data
